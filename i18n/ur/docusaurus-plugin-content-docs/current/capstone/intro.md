# کیپ اسٹون پراجیکٹ

## پراجیکٹ کا جائزہ

یہ کیپ اسٹون پراجیکٹ کورس کے تمام ماڈیولز کے علم کو یکجا کرتا ہے۔ آپ ایک مکمل ایمبوڈیڈ انٹیلیجنس سسٹم ڈیزائن، نافذ، اور جانچیں گے۔

## پراجیکٹ کے مقاصد

اس پراجیکٹ کے اختتام پر، آپ:
- ROS 2 پر مبنی ایک مکمل روبوٹک سسٹم بنائیں گے
- سمولیشن میں اپنے حل کی ٹیسٹنگ کریں گے
- VLA ماڈلز کو روبوٹ کنٹرول کے لیے استعمال کریں گے
- اپنے کام کو دستاویزی شکل میں پیش کریں گے

## پراجیکٹ کی تفصیلات

### منظر نامہ

ایک ہیومنائیڈ روبوٹ بنائیں جو:
1. قدرتی زبان کی ہدایات سمجھ سکے
2. ماحول میں آبجیکٹس کی شناخت کر سکے
3. مینیپولیشن ٹاسکس مکمل کر سکے
4. محفوظ طریقے سے نیویگیٹ کر سکے

### ضروری اجزاء

1. **ROS 2 نوڈز**: کنٹرول اور کمیونیکیشن
2. **سمولیشن**: Gazebo یا Isaac Sim میں ٹیسٹنگ
3. **پرسیپشن**: کیمرا اور سینسر ڈیٹا
4. **VLA انٹیگریشن**: زبان سے عمل

## جائزے کے معیار

- **فعالیت** (40%): سسٹم صحیح کام کرتا ہے
- **کوڈ کوالٹی** (20%): صاف اور قابل دیکھ بھال کوڈ
- **دستاویزات** (20%): مکمل اور واضح
- **تخلیقیت** (20%): جدید حل

## ٹائم لائن

| ہفتہ | سنگ میل |
|------|---------|
| 1-2 | منصوبہ بندی اور ڈیزائن |
| 3-4 | ROS 2 نوڈز کی تعمیر |
| 5-6 | سمولیشن انٹیگریشن |
| 7-8 | VLA ماڈل انٹیگریشن |
| 9-10 | ٹیسٹنگ اور آپٹیمائزیشن |
| 11-12 | دستاویزات اور پریزنٹیشن |

## وسائل

- کورس کے تمام ماڈیولز کا مواد
- سمولیشن ماحول
- پری ٹرینڈ VLA ماڈلز
- مینٹور سپورٹ

## شروع کریں

1. اپنے پراجیکٹ کا انتخاب کریں
2. ڈیزائن دستاویز تیار کریں
3. ترقی شروع کریں
4. باقاعدہ پیش رفت کی رپورٹ دیں

کامیابی کی خواہشات!
