# سمولیشن ماحولیات

## ماڈیول کا جائزہ

یہ ماڈیول روبوٹکس سمولیشن ماحولیات کا جامع تعارف فراہم کرتا ہے۔ سمولیشن روبوٹکس ڈویلپمنٹ میں ایک اہم کردار ادا کرتی ہے، جو آپ کو حقیقی ہارڈویئر کو خطرے میں ڈالے بغیر الگورتھمز کو ٹیسٹ کرنے کی اجازت دیتی ہے۔

## سیکھنے کے مقاصد

اس ماڈیول کے اختتام پر، آپ قابل ہو جائیں گے:
- Gazebo سمولیٹر کو انسٹال اور کنفیگر کرنا
- SDF فارمیٹ میں روبوٹ ماڈلز بنانا
- سمولیشن ورلڈز ڈیزائن کرنا
- ROS 2 کے ساتھ سمولیٹر کو مربوط کرنا

## Gazebo کیا ہے؟

Gazebo ایک اوپن سورس 3D روبوٹکس سمولیٹر ہے جو:

- **فزکس انجنز**: ODE، Bullet، DART، Simbody سپورٹ
- **سینسر سمولیشن**: کیمرے، LiDAR، IMU، GPS، اور بہت کچھ
- **ورلڈ بلڈنگ**: SDF فارمیٹ کے ساتھ پیچیدہ ماحول
- **ROS انٹیگریشن**: ros_gz کے ذریعے ROS 2 کے ساتھ بغیر رکاوٹ کنکشن

## سمولیشن کی اہمیت

### فوائد

1. **محفوظ ٹیسٹنگ**: حقیقی روبوٹ کو نقصان پہنچائے بغیر ٹیسٹ کریں
2. **تیز تکرار**: تبدیلیاں فوری طور پر ٹیسٹ کریں
3. **لاگت میں کمی**: مہنگے ہارڈویئر کی ضرورت نہیں
4. **مشکل حالات**: خطرناک صورتحال کی نقل
5. **متوازی ٹیسٹنگ**: ایک ساتھ متعدد سمولیشنز چلائیں

### استعمال کے معاملات

- الگورتھم ڈویلپمنٹ اور ٹیسٹنگ
- مشین لرننگ ماڈل ٹریننگ
- سسٹم انٹیگریشن ٹیسٹنگ
- تعلیمی مقاصد

## شروع کرنا

### Gazebo انسٹالیشن

```bash
# Ubuntu پر Gazebo انسٹال کریں
sudo apt install ros-humble-gazebo-ros-pkgs
```

### پہلی سمولیشن

```bash
# Gazebo شروع کریں
gazebo --verbose
```

## خلاصہ

سمولیشن روبوٹکس ڈویلپمنٹ کا ایک لازمی حصہ ہے۔ Gazebo ایک طاقتور ٹول ہے جو آپ کو محفوظ اور موثر طریقے سے روبوٹکس سافٹ ویئر تیار کرنے کی اجازت دیتا ہے۔
