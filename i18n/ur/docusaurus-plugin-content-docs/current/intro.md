# فزیکل اے آئی اور ہیومنائیڈ روبوٹکس کورس میں خوش آمدید

جدید روبوٹکس اور مصنوعی ذہانت کی دنیا میں قدم رکھیں۔

## کورس کا جائزہ

یہ کورس آپ کو سکھائے گا:
- ROS 2 کی بنیادی باتیں
- سمولیشن ماحولیات
- NVIDIA Isaac پلیٹ فارم
- Vision-Language-Action ماڈلز

## شروع کریں

بائیں طرف کے سائڈبار سے ماڈیولز منتخب کریں۔
